Tutorial: Cómo crear un proyecto en Adobe Premiere

En este tutorial aprenderas como crear un nuevo proyecto en adobe Premiere

  1. Abrir el software de Adobe Premiere
  2. Selecciona en la parte superior
  3. Nombra el nuevo proyecto
  4. Nuevo proyecto creado

Abrir el software de Adobe Premiere

En tu PC busca el software de Adobe Premiere

Selecciona en la parte superior

En la parte superior izquierda selecciona donde dice nuevo proyecto

Nombra el nuevo proyecto

Pon el nombre que quieras a tu proyecto y listo!

Nuevo proyecto creado

Aparece el nuevo proyecto con el nombre que le diste y ya puedes empezar a trabajar en él
